# Halloween

> A random emoji generator

### Compile

```bash
g++ -O2 -o halloween halloween.cc

```

### Usage

```bash
./halloween
```

Enter the number of emojis you want to get.
And scare yourself!

> This code uses `#include <ctime>` to get a random sequence for the `rand()`
